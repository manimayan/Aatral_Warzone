# Aatral-Warzone
## Soen6441- War Zone group project

### Project Description
War zone game allows the user to select the continent. Once the continent is selected by the user, then the countries belongs to the map will be assigned randomly to each of the player respectively. Initially same number of armies are issued to all the players. Player can deploy the armies in same country to increase the own countryâ€™s army count. Thereafter, during the execution phase he can transfer the armies to his own countries to increase the reinforcement of the country or he can deploy the armies on enemies to attach and conquer the enemy country. Once the player wins in battle then the country will belong to his own and he can add his own armies in the conquered country.
